Свойства итема: Структура и перечень

Item изначально обладает всеми свойствами, но они все имеют значение string.Empty
У Item`а должно быть заполнено поле Name. Иначе он в списке предметов будет пустой строкой.
У Item`а должно быть заполнено поле Category. Иначе он не попадёт в сортировку. У него будет Category.Unknown.

Все свойства должны иметь уникальные имена.
Каждое свойство так же имеет категорию.

Окно редактирования предметов:
При редактировании предмета есть фильтр по категориям для отображения соответствующих свойств. Больше он вроде нигде не используется.
Свойства категории Common отображаются всегда.

Перечень категорий: ItemProperties.Category

	Armor - Броня
    Helm - Шлем
    FullArmor - Броня со шлемом
    Weapon - Оружие
    Ammo - Аммуниция
    Medicine - Медицина
    Mod - Модификация
    Demolition - Взрывчатое в-во
    Sundry - Предмет

=== Common ===
Свойства которые есть у всех итемов

Name: Название. Должно быть заполнено.
Category: Категория. Если не заполнено - будет Unknown и отобразится только в особом фильтре.

Weight: вес
Price: цена
Description: описание
